# Toast MC

### Mod Description:
Adds consumable toast food into minecraft.

![toast](https://github.com/Zircon332/toast-mc/toast.png "Toast Texture")


### Crafting recipe:
> Cook bread in furnace.


### Toast properties:
```
Hunger gain - 8
Saturation - n/a
```
 
 
- In creative, can be found under the foods tab.
